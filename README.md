<h1 align="center">Tcgdex</h1>

<p>Tcgdex is a full stack web application meant to make keeping track of your Pok√©mon card colllection easier. After making an account on our web app, users can search for and add cards to their collections.</p>

React Redux Pokedex
Project made with JAVASCRIPT JS, REACT, STYLED-COMPONENTS, REDUX, POSTGRES


Features:
Add pokemons to your pokedex
Search pokemons by number/name
See pokemon's details
Find pokemons of a type (e.g. all evolutions of pikachu)
